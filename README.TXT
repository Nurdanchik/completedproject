# Документация проекта для хакатона

## Введение

Приветствуем тебя, уважаемый читатель! Эта документация предназначена для того, чтобы ты мог успешно развернуть и запустить проект на хакатоне. Проект был создан специально для участия в данном мероприятии, и мы уверены, что ты найдешь его интересным и полезным.

## Подготовка к запуску

Перед тем как начать, убедись, что у тебя установлен Python на твоем ноутбуке или компьютере. Ты можешь проверить это, введя следующую команду в командной строке:

```bash
python --version
```

Если Python не установлен, рекомендуем посетить официальный сайт [Python](https://www.python.org/) и выполнить установку.

Далее, активируй виртуальное окружение. Если ты используешь Mac, выполни следующие команды:

```bash
python -m venv venv
source venv/bin/activate
```

Если ты используешь Windows:

```bash
python -m venv venv
venv\Scripts\activate
```

## Установка зависимостей

Теперь, когда виртуальное окружение активировано, установи все необходимые зависимости, выполнив следующую команду:

```bash
pip install -r requirements.txt
```

## Запуск проекта

Поздравляем! Ты успешно завершил подготовительные шаги. Теперь можешь запустить проект, используя следующую команду:

```bash
python manage.py runserver
```

Теперь открой свой веб-браузер и перейди по адресу [http://127.0.0.1:8000/](http://127.0.0.1:8000/). Ты увидишь запущенное приложение.

## Завершение работы

После завершения работы с проектом не забудь деактивировать виртуальное окружение, введя следующую команду:

```bash
deactivate
```
